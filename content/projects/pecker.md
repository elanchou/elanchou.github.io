## 项目简介

Pecker 是一款基于人工智能的 RSS 阅读器，为用户提供智能化的内容筛选和推荐服务。

### 主要特性

- 智能内容摘要
- 个性化推荐
- 多源 RSS 订阅管理
- 优雅的阅读体验

### 技术栈

- SwiftUI
- Core ML
- ChatGPT API
- Core Data
